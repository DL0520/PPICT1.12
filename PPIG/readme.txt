Some of the files and code in this folder that are primarily part of the PPIG subnet.

Cross-talk_graph/PPI_graph: Some basic files needed for graph coding input, such as adjacency matrix, initial node features, etc.
FEGS: Note that this needs to be performed using matlab. Mainly used to extract protein sequence features and return them in XX.m format.
SDNE: Encoding PPIs graph and extracting corresponding protein features.
Linear_VAE: Encoding cross-talk graph and extracting corresponding protein features.
