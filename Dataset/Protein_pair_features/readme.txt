Protein features extracted from cross-talk graph structure information using Linear _ VAE in PPIG.

Cross_talk_gcn_features128.npy: protein(128-dim) features extracted by Linear_VAE, which origin node features are extracted by SDNE(PPIs graph). - PPICT model
Cross_talk_gcn_features128_FEGS.npy: protein(128-dim) features extracted by Linear_VAE, which origin node features are extracted by FEGS. - W_CT model
PPIs.npy: protein(128-dim) features extracted by SDNE from PPIs graph. - W_PPI model
Protein_num.txt: Feature corresponding protein number.
